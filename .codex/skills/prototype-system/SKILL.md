# prototype-system

Follow the same workflow as `.claude/commands/prototype-system.md`.

Read that command file first, then execute it. Create:

- `prototypes/<system-name>/README.md`
- one prototype artifact: `conversation-mockup.md`, `index.html`, or
  `system-spec.md`

Keep the scope to one believable demo interaction. Do not build a full
production system.
